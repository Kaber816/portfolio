Brain Controlled Rover Research

The second project that I have worked on during my time as an undergraduate research is a brain signal controlled rover. I started this project in August of 2023 by myself as a sophomore and have since started working with a team of other people with backgrounds in computer science, mechanical engineering, and anatomy. The goal of this project is to create rover that can use signals from the brain to perform actions like driving forward and turning.  To do this, we are currently using an Emotiv Insight headset that utilizes machine learning that we then link to an Arduino microcontroller using a special software called HITIBrain. 
In my sophomore year I contributed to this project mostly by prototyping and setting up the code we need to interface the headset with our rover, I also worked on creating some stuff to design our own rover with better functionality as well. We started the project using a prebuilt robotics competition rover, but after figuring out our use cases and what we would need to make the project work I decided that we would have to overhaul the bot to get the functionality we wanted. To do this I went through schematics online of the existing bot and found a way to link the systems with an external motor driver and 18650 battery pack which would allows us to fully customize the bot however we wanted. To do this I used the techniques and lessons I learned in my CPRE 288 embedded systems class which involves creating code and using sensors on an IRobot device. Some important skills I have learned from this project include working with microcontrollers, using Arduinos, and creating embedded systems. 
This project has important applications outside of our classroom because we are figuring how to develop systems that are able to utilize brain waves as control inputs. This will pave the way for new ways to control devices ranging from simple things like lights to more ambitious things like controlling a bionic hand. While we are just beginning our research using these devices and techniques, I hope to continue and improve upon the way we use them for the next couple semesters as I keep working on it.
