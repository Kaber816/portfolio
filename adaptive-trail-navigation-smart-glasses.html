Adaptive Trail Navigation Smart Glasses

Individuals with visual impairments often encounter significant challenges when safely navigating their environments. While existing mobility aids such as white canes and guide dogs provide valuable assistance, they have limitations in detecting elevated obstacles and conveying detailed spatial information. To address these gaps, my senior design team at Iowa State University is developing Adaptive Trail Navigation Smart Glasses with BAE systems and Adaptive Adventures that leverage sensor technologies to improve spatial awareness and promote independent mobility. 
My role in this project so far is on the embedded systems and 3D design sides of this project. Because of my previous experience working with designing and working on embedded systems in classes, as well as my personal focus on it for my career, a lot of my personal contributions to our work have dealt with I2C programming in python and c++. This has been an amazing learning experience for me because it has allowed me to apply the skills I've learned in classes in a more professional scenario where I'm working with a team to design a prototype from the ground up. 
As for the 3D design part of this project, while not a new concept to me, this has been the first time that I've actually developed with and used CAD for a serious project. This has been a really exciting experience for me since I've been 3D printing stuff for years, but have never really worked with a serious CAD program like OnShape. While my design skills are often limited and I need a lot of help from resources and tutorials while doing it, I feel like this has been a valuable skill that I've been able to build during our project so far. While our prototype is not complete yet this semester. I hope to work more on these skills throughout the year and deliver a finished product that I'm proud of!
